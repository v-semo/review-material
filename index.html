URL Parameters

The functionality of V-SeMo can be customized using URL parameters. These are key=value pairs appended to the URL following a question mark (?). How to change the language of the user interface or enable extra buttons like "Set Sectors", "Complete", "Direction", and "Joystick":

  • Sector model with default interface:

    https://v-semo.github.io/review-material/sp2.html

  • Set language to English or German:

    https://v-semo.github.io/review-material/sp2.html?lang=en
    https://v-semo.github.io/review-material/sp2.html?lang=de

  • Enable "Set Sectors" button:

    https://v-semo.github.io/review-material/sp2.html?showSetSectors=1

  • Enable "Complete" button:

    https://v-semo.github.io/review-material/sp2.html?showAutoComplete=1

  • Enable "Direction" button:

    https://v-semo.github.io/review-material/sp2.html?showChangeDirection=1

  • Enable "Joystick":

    https://v-semo.github.io/review-material/sp2.html?showChangeStartPoint=1

  • Combine multiple options (use ampersand &):

    https://v-semo.github.io/review-material/sp2.html?showSetSectors=1&showAutoComplete=1
